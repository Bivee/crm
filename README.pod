=head1 Bivee CRM

Bivee CRM(under high development) is a simple CRM solution implemented using 
Perl 5.

<p><img src="bivee-crm.png" alt="Bivee CRM" title="Bivee CRM screenshot [under dev]" width="400px"></p>

=head3 Who can use it?

Anyone can download a copy of this software, install on server and use!


=head3 What do I need to take to install then?

=over

=item Perl 5.10+

=item Mojolicious 4+ (and some plugins)

=item ObjectDB ORM

=item MySQL 5

=back


=head3 How can I test/run?

You can use C<morbo> that is an development perl server for mojolicious to run
L<Bivee::CRM> on your machine:

    $ morbo crm/index.pl

or you can use C<hipnotoad> for production deployment.
